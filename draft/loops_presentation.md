## Code Example - Let’s Make the Computer Clap!
- **Visual**: Python code snippet:
   ```python
   for i in range(5):
       print("Clap!")
   ```
- **Speech**:
   - "Here’s our first trick! We’re going to make the computer clap for us. This little bit of code is our magic wand!"
   - "See this word ‘for’? It’s like saying ‘let’s go around in a circle.’ And ‘range(5)’ means we’ll do it 5 times. And then ‘print(“Clap!”)’ is telling the computer to clap each time."
   - "Now, let’s try it out! Watch the computer clap 5 times!"
